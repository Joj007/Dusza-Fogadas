# **Szerepkörök**

## Fogadó
- látja a saját összegét
- nyílt játékok böngészése
- fogadás nyílt játékokra (1 alany-esemény párosra)
- *múltbéli fogadásainak áttekintése*

## Szervező
- játék létrehozása
- játék lezárása, eredmények megadása

## Admin
- Játékok törlése
- Felhasználók törlése
